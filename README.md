# Business Operations Dashboard

This dashboard analyzes company performance across departments, products, suppliers, and employees using SQL and Power BI.

## Key Features
- Revenue and profit margin KPIs
- Product-wise revenue and feedback
- Employee performance by department
- Supplier inventory distribution
- Projected growth metrics

## Tools Used
- SQL (CTEs, views, triggers, stored procedures)
- Power BI / Excel
- Data cleaning and visualization
